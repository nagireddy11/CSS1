What is CSS?

  1. Css stands for Cascading Style Sheet

  2. CSS describes the how HTML elements are to be displayed on screen,paper,or in other media.

  3. Css is not necessary CSS saves a lot of work. it can control the layout of multiple webpages all at oncde

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML document in 3 ways

  • Inline-by using the style attribute inside HTML elements1
  • internal-by using a style element in the head section
  • External-by using a link element to link to an external CSS file.
  • The most common way to add CSS ,is to keep the styles in a external CSS files .However in this tutorial we will use the inline and internal styles. because this is easier to demonstrate and easier for you to try it yourself
